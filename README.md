## Example

### Input

`count += 1`

### PyLine Tutor Output

Line of code

`count += 1`

Read as

Increase count by one.

Word-by-word explanation

• `count` → the variable storing a number

• `+=` → adds a value to the current number

• `1` → the number being added

Meaning of the line

This line increases the value inside `count` by 1.

Importance of the line

This is commonly used in loops and counters. Without it, the program would never keep track of how many times something happened.

## Another Example

### Input

`def greet(name):`

### PyLine Tutor Output

Line of code

`def greet(name):`

Read as

Create a function called greet that uses a name.

Word-by-word explanation

• `def` → tells Python you are creating a function

• `greet` → the name of the function

• `(` → starts the input section

• `name` → the value the function will receive

• `)` → closes the input section

• `:` → starts the function block

Meaning of the line

This line creates a reusable function named `greet`.

Importance of the line

Functions help organize code and avoid repeating the same instructions many times.
